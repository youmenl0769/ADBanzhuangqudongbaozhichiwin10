 ## 使用说明

 1. **下载驱动**：从本仓库下载最新的ADB驱动安装包。
 2. **安装步骤**
    - 双击运行下载的安装程序。
       - 按照屏幕提示进行操作，通常只需要点击“下一步”继续。
          - 如果系统提示需要管理员权限，确认以继续安装。

             3. **验证安装**：
                - 打开命令提示符或PowerShell，输入 `adb devices`。
